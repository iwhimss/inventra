## Kök Neden Analizi

### Title Bar Bug

`main.dart`'ta `WindowOptions.titleBarStyle = TitleBarStyle.hidden` ile native Windows title bar'ı tamamen gizleniyor.
Bu yüzden tüm ekranların kendi `CustomTitleBar` widget'ını dahil etmesi gerekiyor.

**Sorun:** `ServerConnectScreen` (`inventra_app/lib/features/auth/screens/server_connect_screen.dart`) bu widget'ı kullanmıyor.
- `LoginScreen` → `CustomTitleBar` var ✅
- `ServerConnectScreen` → `CustomTitleBar` yok ❌

**Çözüm:** `ServerConnectScreen.build()` içindeki `Scaffold.body`'yi `Column` ile sarmalayıp üste `CustomTitleBar()` eklemek.

```dart
// Önce:
body: Center(
  child: SingleChildScrollView(...)
)

// Sonra:
body: Column(
  children: [
    const CustomTitleBar(),
    Expanded(
      child: Center(
        child: SingleChildScrollView(...)
      ),
    ),
  ],
)
```

### 🔲 FAZA 1 — VDS/VPS Kurulum Kılavuzu

**Hedef:** Bir sysadmin veya geliştirici'nin sıfırdan bir VDS üzerinde `inventra_server`'ı nasıl kuracağını adım adım açıklayan döküman.

**Oluşturulacak dosya:** `docs/vds-deployment.md`

**Kapsam:**

**Bölüm 1 — Gereksinimler**
- İşletim sistemi (Ubuntu 22.04 LTS önerilen)
- Dart SDK kurulumu (pub.dev resmi talimatları)
- SQLite bağımlılıkları (`libsqlite3-dev`)
- Port durumu (5000, ya da özelleştirilmiş)

**Bölüm 2 — Proje Kurulumu**
- Git clone işlemi
- `dart pub get` ile bağımlılık kurulumu
- `dart run bin/server.dart --setup` ile ilk kurulum (interaktif wizard: işletme adı, admin ID/şifre, port, network erişimi)
- `config.json` ve `inventra.db` yapısı

**Bölüm 3 — Firewall Ayarları**
- UFW ile port 5000 açma
- Cloud provider (Hetzner, DigitalOcean vb.) panel'inde firewall kural ekleme

**Bölüm 4 — Sunucu Çalıştırma**
- Manuel: `dart run bin/server.dart` / `./start.sh`
- `systemd` servisi kurulumu (örnek unit dosyası dahil)
- Otomatik başlatma: `systemctl enable inventra-server`
- Log takibi: `journalctl -u inventra-server -f`

**Bölüm 5 — Flutter Uygulaması Bağlantısı**
- VDS IP:PORT formatında sunucu adresi girişi
- Eşleme (pairing) onayı: sunucu admin panelinden cihaz onaylama
- Bağlantı testi

**Bölüm 6 — İsteğe Bağlı: Reverse Proxy (Nginx)**
- HTTPS için Nginx reverse proxy kurulumu
- Certbot ile SSL sertifikası (Let's Encrypt)
- Nginx konfigürasyon örneği

**Bölüm 7 — Güncelleme Prosedürü**
- `git pull` + `dart pub get` + `systemctl restart inventra-server`

- [x] **1.1** `docs/` klasörü yok — önce kontrol et, yoksa oluştur
- [x] **1.2** `docs/vds-deployment.md` dosyasını yaz (Türkçe başlıklar + kod blokları)
